# Klasyfikacja Ryzyka Zawodowego BHP

Projekt zaliczeniowy. Algorytm ocenia ryzyko zawodowe na podstawie metody RISK SCORE. Zastosowano prostą sieć neuronową (MLP) klasyfikującą ryzyko jako Akceptowalne (0) lub Nieakceptowalne (1).

# Jak uruchomić projekt:
1. Pobierz repozytorium.
2. Zainstaluj zależności: `uv pip install numpy pandas`
3. Uruchom skrypt: `uv run main.py`
